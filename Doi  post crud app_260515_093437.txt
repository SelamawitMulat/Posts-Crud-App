Design a modern mobile Flutter CRUD Posts App UI.

The app is a Post Management application using a clean and minimal design style.

Main color theme:
- Blue and white
- Soft shadows
- Rounded corners
- Modern Flutter Material 3 design

Screens to design:

1. Splash Screen
- App logo icon
- App name: "Posts CRUD App"
- Minimal loading animation

2. Home Screen
- AppBar title: "Posts"
- Search icon in AppBar
- Floating Action Button with "+" icon
- List of post cards

Each post card should contain:
- Post title
- Short post body text
- Edit icon button
- Delete icon button
- Clean card layout with spacing and shadows

3. Add/Edit Post Screen
- Title text field
- Body multiline text field
- Save button
- Clean form layout
- Back button in AppBar

4. Loading State UI
- Circular loading indicator
- Minimal centered design

5. Error State UI
- Error illustration
- Error message text
- Retry button

Design requirements:
- Mobile-first layout
- Android Flutter style
- Use modern typography
- Consistent spacing and padding
- Smooth and professional appearance
- Use Material 3 components
- Beginner-friendly but visually impressive
- Clean UI suitable for a university assignment

Include:
- Icons
- Buttons
- Card components
- Floating Action Button
- Empty state illustration

Make the design look similar to modern productivity apps.
